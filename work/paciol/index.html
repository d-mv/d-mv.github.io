Overview 01

A high-fidelity ledger for personal finance.

Named after the father of accounting, Paciol is a personal finance application built to be simple in use, with full feature power under the hood. It applies formal double-entry principles to personal data, ensuring that every cent is accounted for across accounts, categories, and portfolio positions.

The application is built with pure JavaScript, Vue and PrimeVue components (for rapid development to be later replaced by vanilla Vue/CSS ones). It handles everything from daily transaction tracking to complex recurring payments and investment performance, all synced in real-time via Supabase.

The goal was to build a tool that feels more like a professional workstation than a mobile app — a place where data integrity and precision are the primary features.

Outcome 03

Professional tools for personal use.

Paciol successfully bridged the gap between personal finance tracking and professional accounting. The use of Supabase allowed for a production-grade backend with minimal operational overhead.

The project proved that even for personal utility, investing in “heavy” architectural principles like double-entry pays off in long-term data reliability and insight.
