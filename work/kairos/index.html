Overview 01

A task engine built to be controlled by humans and understood by machines.

Kairos is more than a todo list; it is a task workstation - to be used in mix-type workflows. The name, derived from the ancient Greek for “the opportune moment,” reflects the project’s goal: helping users focus on what matters now.

What sets Kairos apart is its native integration with the Model Context Protocol (MCP). By exposing a secure, authenticated MCP server, Kairos allows AI assistants like Claude to read, create, and manage tasks directly, turning a passive list into an active, collaborative environment.

The project is built using latest technologies - Vue, TypeScript, Elysia and Bun, for performance and ease of further development.

A unified platform for action.
